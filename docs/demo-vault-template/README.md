# OMSB Demo Vault Template

This directory is a **documentation-only, sanitized demo fixture** for screenshots, GIFs, and walkthroughs.

It is **not** a runtime fixture, golden output, or verification authority.

Use it when you want to:
- record a privacy-safe OMSB demo
- stage screenshots for README/docs/release pages
- demonstrate `/omsb init`, `/omsb terminology`, or `/omsb plugin-settings`

Do not put real notes, real folder names, usernames, secrets, or personal vault structure into this template.
