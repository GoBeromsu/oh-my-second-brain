# OMSB Shot List

This document defines the **first public demo** for OMSB.

Before publishing any screenshot or GIF, review:
- [docs/demo-guide.md](demo-guide.md)

## First public demo sequence

### Shot 1 — Vault-local setup context
- Show a sanitized demo vault only
- Show the generic guideline folder and safe demo structure
- Goal: establish that OMSB is vault-scoped

### Shot 2 — `/omsb init`
- Show guideline folder selection
- Show folder/frontmatter guideline mapping
- Show managed plugin registration
- Goal: explain mapping + initialization

### Shot 3 — `/omsb terminology`
- Show an explicit routing outcome or an `Inbox` fallback
- If routing is ambiguous, show proposal behavior rather than guessing
- Goal: demonstrate `explicit | inbox | propose`

### Shot 4 — `/omsb plugin-settings`
- Show inspection of a managed plugin's `data.json`
- Show a safe, guideline-explicit setting update or an approval-needed optimization proposal
- Goal: explain the managed-plugin boundary
